### Quick Start
1. Clone the repository:
   \`\`\`sh
   git clone https://github.com/timedilationv2/trikiwiki.git
   cd trikiwiki
   \`\`\`
2. Navigate to the wiki directory:
   \`\`\`sh
   cd wiki/
   ls -la
   \`\`\`
3. Open a file for editing:
   \`\`\`sh
   nano [file].md
   \`\`\`
